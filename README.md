
# silver-horn-opencl
The library enables .NET/Mono applications to take full advantage of the OpenCL framework.

#### Library based on
- [Cloo](https://sourceforge.net/projects/cloo/)
- [OpenCLTemplate](https://code.google.com/archive/p/opencltemplate/)
